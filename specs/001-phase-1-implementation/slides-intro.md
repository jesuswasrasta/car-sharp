# Welcome to CarSharp

## Phase 1: Minimal Fleet Management

In this phase, we explore the most basic form of state management.

### The OOP Approach
- Encapsulation of a mutable list.
- Methods that change internal state (`void`).
- Verification via Example-Based tests (Facts).

### The Functional Approach
- State as an immutable value (record).
- Pure functions that return new state instances.
- Verification via Property-Based tests (FsCheck).
